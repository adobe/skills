# Classic Archive

The v0.1 default chassis, preserved as one option among several. Editorial-archival presentation — numbered sections, sticky top nav, masthead with metadata stamp, rigorous horizontal grid.

## Intent

Feels like a design document from an archive — a brand board that could sit on a shelf between Pentagram annual reports and *The Gentlewoman* issues. Commits to seriousness without being cold. Appropriate for brands whose subject matter is itself archival, editorial, institutional, or durable (publishers, law firms, museums, artisan goods, fine-press printers, long-run manufacturers).

## Navigation

Sticky top bar. Horizontal link list, one line. Brand name on the left, section links center-right, a metadata stamp on the far right ("VOL 01 · BRAND BOARD · YYYY-MM-DD"). Nav background matches the page body; a 1px ink hairline separates nav from content.

## Section order

Canonical top-to-bottom:

1. Masthead
2. Palette
3. Typography
4. Voice (do / don't)
5. Motifs
6. Components
7. Photography direction
8. Personas
9. Logo variants
10. Spacing & shape (if present)

## Hero shape

Masthead with a three-part structure:

- Large brand name, serif or slab display register.
- One-line tagline directly below, smaller, in the editorial register.
- Metadata stamp below the tagline in mono: location · year · batch / edition number.

No hero image. The masthead is text-only, the way a book's title page is.

## Eyebrow / label conventions

Numbered section eyebrows: `§ 01 · Palette`, `§ 02 · Typography`. The section number is the eyebrow; the section title is a standard-size heading immediately below. Eyebrow font is mono, all caps, tracked out, small (11–12px).

## Spacing rhythm

Airy and rigid. Sections separated by generous vertical rule (96px+). Horizontal gutters consistent. No overlap, no negative margins. Every element aligns to a column grid.

## Typography register

Balanced display + body. Display carries mastheads and section titles; body carries voice examples, motif descriptions, and persona text. Monospace appears only in eyebrows, metadata, and component specs. Italic serif used sparingly on single accent words within body copy.

## Motif placement

Inline demo tiles in a card grid. Each motif gets a fixed-size demo box (~240×180px), the motif rendered inside it, name + description below. The grid is horizontal, 3 or 4 columns wide.

## What this chassis is NOT

- Not a dashboard — no meters, gauges, status chips, or progress indicators
- Not a magazine — no pull quotes, column text, or issue numbering
- Not a pinboard — no rotation, overlap, or absolute positioning
- Not a SaaS landing page — no oversized sans-serif hero, no two-CTA row

## Example reference

A Pentagram annual report. A volume of *MacGuffin*. A Rizzoli monograph's appendix.
