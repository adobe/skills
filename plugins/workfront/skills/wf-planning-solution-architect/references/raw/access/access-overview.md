# Adobe Workfront Planning access overview

Source: https://experienceleague.adobe.com/en/docs/workfront/using/adobe-workfront-planning/adobe-workfront-planning-access/access-overview
Last update: Wed Apr 01 2026

## Access requirements

### Adobe Workfront package
- Any Workfront and Planning package, OR any Workflow and Planning package
- For connectable record types: Any Workfront + a Planning package; Any Workflow + Planning Prime or Ultimate
- For global record types: Any Workfront + Planning Plus; Any Workflow + Planning Prime or Ultimate

### Adobe Workfront platform
Must be onboarded to Adobe Unified Experience. Only users in Adobe IMS can be granted permissions and added to Planning fields.

### Adobe Workfront license
- Any license: view WFP information
- Standard: create workspaces and views

### Access level configurations
No access level controls for WFP specifically.

### Object permissions
- View or higher: access workspaces, record types, views you didn't create.
- Contribute or higher: edit workspaces and record types, create/edit/delete record types and records.
- Contribute or higher on views you didn't create: edit, delete, share.
- System Admins can manage workspaces they didn't create.
- System Admins cannot access views they didn't create.

### Layout template
- Standard users and System Admins have Planning enabled by default.
- Light/Contributor users need layout template with Planning in:
  - Main Menu
  - Left panel of projects, portfolios, programs

## Share Planning area via layout template

Admin can add Planning to:
- Main Menu
- Landing page
- Left panel for projects, portfolios, programs
- Pins

Steps: Main menu → Setup → Interface → Layout Templates → open/create → assign users.

## Assign licenses
Configure access level while editing/creating users.

## Configure access level
No Workfront access level controls for WFP.

## Grant permissions
Permissions on:
- Workspaces
- Record types
- Views

Workfront license type interacts with WFP permissions to determine view/contribute/manage capability.
