# Edit record types

Source: https://experienceleague.adobe.com/en/docs/workfront/using/adobe-workfront-planning/adobe-workfront-planning-architecture/edit-record-types
Last update: Thu Jan 15 2026

Editable on the Edit record type box:
- Appearance tab: name, description, icon color/shape
- Cross-workspace settings tab (sysadmin only): set as connectable and/or global

After save:
- Connectable record icon displays if "connect from other workspaces" enabled
- Global record icon displays if "add to other workspaces" enabled

Permissions: Manage on workspace. Sysadmins have all.
