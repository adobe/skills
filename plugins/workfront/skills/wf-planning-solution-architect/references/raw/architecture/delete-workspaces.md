# Delete workspaces

Source: https://experienceleague.adobe.com/en/docs/workfront/using/adobe-workfront-planning/adobe-workfront-planning-architecture/delete-workspaces
Last update: April 1, 2026

## Critical considerations
- Deleting a workspace deletes ALL record types, records, fields, and views.
- Deletion is permanent. Cannot be recovered.
- Recommendation: recreate critical artifacts in another workspace before deletion.

## Permissions
- Standard license
- Manage permissions on workspace
- Sysadmins on all

## Confirmation
- Requires typing "delete" (case-insensitive) before clicking Permanently delete.

## Keyboard shortcut for global search
- Ctrl+K (Win) / ⌘+K (Mac)
