# Configure the Settings area of a record type

Source: https://experienceleague.adobe.com/en/docs/workfront/using/adobe-workfront-planning/adobe-workfront-planning-architecture/configure-record-type-settings
Last update: April 1, 2026

Settings page is an alternative entry point for record type configuration (same scope as Edit's Cross-workspace settings tab). Currently houses:
- Cross-workspace settings section (default):
  - Allow adding this record type to other workspaces (global flag)
  - Allow connecting to this record type in other workspaces (connectable flag)
- Both off by default.

Permission: Manage on workspace. Standard license.
