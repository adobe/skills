# Edit workspaces

Source: https://experienceleague.adobe.com/en/docs/workfront/using/adobe-workfront-planning/adobe-workfront-planning-architecture/edit-workspaces
Last update: Thu Jan 15 2026

## What you can edit
- Workspace name, description, icon
- Sections: add, reorder (drag or Move up/down), delete (must move record types to another section first)
- Record types: edit, delete, drag-drop between sections within same workspace
- Share workspace via Share button

## Section deletion
- When deleting a section, all record types must be moved to a different section.

## Notes
- All edits visible to anyone with View+ on workspace.
- Permission: Manage on workspace.
