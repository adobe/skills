# Create workspaces

Source: https://experienceleague.adobe.com/en/docs/workfront/using/adobe-workfront-planning/adobe-workfront-planning-architecture/create-workspaces
Last update: Thu Oct 16 2025

Workspaces are centralized locations for teams to plan work. A workspace is a collection of record types representing a team's work lifecycle.

## Access requirements

- Workfront package: Any Workfront + Planning, or Any Workflow + Planning
- License: Standard
- Object permissions: Manage permissions on workspace (sysadmins have on all)

## Creating a workspace

1. Main Menu → Planning.
2. Optional (admin): toggle "My workspaces" or "Other workspaces".
3. Click Create workspace.
4. Optional: Preview templates:
   - Basic: Marketing Management
   - Advanced: Marketing Management
   - Enterprise: Marketing Management
   - Sales management
   - Product management
5. Click "Use template" or "New workspace" from scratch.
6. Rename workspace (click name in header, Enter).
7. Optional: rename sections (Operational Record Types, Taxonomies) via More menu → Rename. Sections can be renamed by anyone.
8. Add record types via "Add record type" in any section.
9. New workspace card displays in Workspaces I'm on tab. Creator saved as Owner.

Note: for users mid-transition to IMS, workspaces created by Workfront-only users (not IMS) display as created by "System".
